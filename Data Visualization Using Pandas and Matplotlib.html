Customizing Graph Appearence¶

Marker¶

The plot method takes in optional key word argument, marker, which lets you indicate that you are interested in adding markers to the plot and also what kind of marker you would like. Below is an example with a circle marker and all markers available are shown here.

In [10]:
data_time = df.groupby('month')['ext price'].mean().reset_index()
In [11]:
data_time.plot(x='month', y='ext price',kind='line',marker='o')
Out[11]:
<matplotlib.axes._subplots.AxesSubplot at 0x1a268234a8>
Linestyle¶

When you have two lines in the same axes, you may need to change the line style to distinguish them. Like marker, linestyle is another optional key word argument in plot method. Below is an example using dotted lines and all available line styles are shown here.

In [12]:
data_time.plot(x='month', y='ext price',kind='line',marker='o', linestyle=':')
Out[12]:
<matplotlib.axes._subplots.AxesSubplot at 0x1a26b04cc0>
Color¶

Color is important when drawing a plot. It can be used to distinguish different categories or just to make the graph more pretty. When choosing a color, we suggest you this reference. Like the other two, color is also an optional key word argument in plot function.

In [13]:
